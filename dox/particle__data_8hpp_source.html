23 /** \file
24  * Particles property access.
25  *
26  * This file contains everything related to particle properties. If you want to
27  * add a new property to the particles, it is probably a good idea to modify
28  * Particle to give scripts access to that property. You always have to modify
29  * two positions: first the print section, where you should add your new
30  * data at the end, and second the read section where you have to find a nice
31  * and short name for your property to appear in the Python code.
32  */
