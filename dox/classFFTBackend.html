Detailed Description

template<typename FloatType>
class FFTBackend< FloatType >

API for the FFT backend of the P3M algorithm.

Any FFT backend must implement this interface. The backend can read some members of p3m_data_struct but can only modify the FFT buffers in P3MFFTMesh.
