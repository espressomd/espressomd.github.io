Detailed Description

CUDA implementation of SystemInterface.

When data is synchronized between host and device memory, a subset of the Particle struct is copied from each particle on the host to the corresponding CUDA_particle_data struct on the device via EspressoSystemInterface::gatherParticles(). Once the transfer is complete, the particle AoS on the device is copied (or "split") to a SoA via EspressoSystemInterface::split_particle_struct().
